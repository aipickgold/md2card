# CardMD · Roadmap

> 规划可能随社区反馈调整。有建议欢迎提 [Issue](https://github.com/aipickgold/md2card/issues)。

## 近期(6 周内)

- [ ] **截图 / demo GIF 上传** — README + docs 里的 "截图建设中" 占位替换
- [ ] **新增 5-10 个主题** — 重点补充长图(9:16)和方形(1:1)场景
- [ ] **v1.1.0 Release** — 打包当前稳定版本 + 发布说明
- [x] **Cover 系列重设计 + 3 开源字库** — 2026-04 已上线

## 中期(6 个月内)

- [ ] **主题市场(社区贡献)** — 用户自制主题审核后上架,贡献者署名
- [ ] **自定义主题编辑器** — 可视化调整颜色/字体/布局生成新主题
- [ ] **移动端 Web 适配** — 手机也能编辑和导出
- [ ] **AI 选题助手** — 按话题自动生成多页卡片草稿
- [ ] **动态卡片(GIF / 视频)导出** — 文字逐字出现的短视频

## 长期(2026-2027)

- [ ] **API 批量生成** — 自动化工作流调用
- [ ] **自托管 Docker 镜像** — 企业私有部署
- [ ] **小红书官方数据分析对接** — 看哪些主题转化率最高
- [ ] **其他平台适配** — 微博九宫格、Instagram、知乎专栏封面
