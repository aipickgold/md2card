# md2card · 10 个使用场景示例

## 1. 读书笔记(5 页)

```
用户:把《xxx》这篇读书笔记切成 5 张小红书卡,知识蓝主题

Skill:
  - 主题:know-blue
  - 结构:封面 → 核心观点 → 3 个 takeaways → CTA
  - 用 `---` 分 5 页
  - 生成 ZIP 链接
```

## 2. 产品测评(8 页)

```
用户:测评 iPhone 16,做成小红书,知识蓝,每页一个卖点

Skill:
  - 主题:know-blue
  - 8 页 = 1 封面 + 5 卖点 + 对比表 + CTA
  - 加粗关键数据
```

```
Skill:
  - quote-black 主题(大衬线居中)

  - card_1.png 下载
```

## 4. 财经快讯

```
Skill:
```

```
Skill:
```

## 9. 对比清单

```
用户:做个"程序员 vs 产品经理"对比清单,4 页

Skill:
  - mag-gray 主题(杂志感)
  - 4 页 = 封面 + 2 对比 + 结论
```

## 10. 首图 + 多页正文

```
用户:第一张做封面,剩下 4 张用知识冰蓝,主题"AI 工具"

Skill:
  - cover-sunset(首图)+ know-blue(内容)
  - 5 页
  - 封面 + 4 内容
```

## 常见反馈

- **"自动分页不合理"** → 手动插入 `---` 指定分页点
- **"主题不喜欢"** → 浏览 [aipickgold.com/card-gen/themes](https://aipickgold.com/card-gen/themes) 换一个
- **"图片很大下载慢"** → 建议每张图压缩到 500KB 以下
- **"没找到触发词"** → 试"小红书"/"图文卡片"/"做成卡片"
