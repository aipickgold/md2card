# CardMD · 常见问题

完整版请访问 [aipickgold.com/card-gen/help](https://aipickgold.com/card-gen/help)。

## 基础使用

**Q: 怎么开始?**
A: 打开 [aipickgold.com/card-gen](https://aipickgold.com/card-gen),粘贴 Markdown,选主题,点底部「全部导出」下载 ZIP。

**Q: 简化 vs 高级模式?**
A: 简化 = 紧凑侧栏 + 6 个常用控件,快速出图。高级 = Stitch 风三栏工作区,精细控制水印 / 画布尺寸 / 字体 / 字重。

**Q: 内容太长装不下?**
A: CardMD 自动分页。不用管,直接导出就是多张卡片。想精确控制在哪里断开,插 `---` 即可。

## 主题

**Q: 55 个主题怎么挑?**
A:
- 干货 → 知识卡片 / 教程步骤
- 品牌 → 全框品牌 / 封面
- 长文 → 极简白底 / 杂志极简
- 情感 → 莫兰迪
- 金句 → 金句观点
- 资讯 → 资讯新闻

详见 [themes.md](./themes.md)。

**Q: 怎么切换字体?**
A: Pro 订阅解锁 3 个开源字库(思源宋体 / 霞鹜文楷 / 得意黑)。高级模式 → 属性面板 → 文本 → 字体。

## 导出

**Q: 导出尺寸多大?**
A: 3:4 = 1080×1440(小红书推荐) / 1:1 = 1080×1080 / 9:16 = 1080×1920,@2x 高清。

**Q: 字体显示不对?**
A: 开源字体需要联网加载。刚进页面等 2 秒再导出,或切换一次主题刷新。

**Q: 批量导出多张?**
A: 底部「全部导出 N 页」→ ZIP 下载,文件名 card_1.png / card_2.png...

## 水印

**Q: 免费版水印能去掉吗?**
A: 不能。Pro 订阅解锁自定义水印(文字 + 5 位置)或完全移除。
