## 挑主题建议

- **干货类** → 知识卡片 或 教程步骤(清晰的层级结构)
- **品牌宣传** → 全框品牌 或 封面首页(视觉冲击强)
- **长文深度** → 极简白底 或 杂志极简(无装饰让文字说话)
- **情感生活** → 莫兰迪(低饱和和谐感)
- **硬核技术** → 暗夜模式(深色专业)
- **金句刷屏** → 金句观点(超大衬线 + 装饰引号)
- **资讯快讯** → 资讯新闻(红绿表格 + 时间戳)
- **首图吸睛** → 封面首页(3 层装饰 + 大标题)

## 字体选择

Pro 订阅解锁 3 个开源中文字库(SIL OFL 协议):

- **思源宋体** — 文艺、书卷气 → 适合金句/长文
- **霞鹜文楷** — 手写、温度 → 适合生活/情感
- **得意黑** — 大字有态度 → 适合标题/口号

免费版使用系统字体(苹方/宋体/楷体)。
