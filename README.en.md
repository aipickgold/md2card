## ✨ Why CardMD

Xiaohongshu posts need content sliced into 3:4 image cards. Traditional manual layout takes 1-2 hours per post. **CardMD compresses that to 30 seconds.**

- **Auto-pagination** — long text splits into cards based on content density
- **55+ themes · 10 series** — branded / magazine / knowledge / minimal / morandi / dark / news / quote / tutorial / cover
- **Stitch-inspired advanced workspace** — left template library + center canvas + right property panel
- **3 open-source Chinese fonts** — Source Han Serif · LXGW WenKai · Smiley Sans (SIL OFL)
- **PNG + ZIP export** — @2x HD, one-click batch
- **Privacy-first** — html2canvas renders in your browser, nothing uploaded

## 🚀 Quick Start

Just open the website — no install, no signup:

```
1. Open https://aipickgold.com/card-gen
2. Paste Markdown
3. Pick a theme (55+) → auto-paginate
4. Export single PNG or ZIP bundle
5. Upload to Xiaohongshu image post
```

Advanced mode (Pro):

```
- Toggle "简化 / 高级" at top
- Left: browse templates by 8 categories
- Center: multi-page canvas with thumbnails + zoom
- Right: watermark + canvas W/H + radius + padding + font + weight
```
